# analysis - Real-time and offline analysis for biofeedback and generation of reports

This is a repository of different analysis modules and scripts (mostly in Python) for the WheelSims project. It will contain scripts that analyse wheelchair propulsion kinematics and kinetics in real time, to generate biofeedback or automated reports for clinicians.

Developers should install these modules to be able to run:

```
sh check_code_quality
```

before asking a code review.

```
conda install -c conda-forge black pytest pylint mypy
```
